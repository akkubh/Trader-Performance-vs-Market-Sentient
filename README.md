# Trader Performance vs Market Sentiment Analysis

## Overview

This project analyzes the relationship between Bitcoin market sentiment and Hyperliquid trader performance using the Bitcoin Fear & Greed Index and historical trading data.

## Dataset

- Bitcoin Fear & Greed Index
- Hyperliquid Historical Trader Data

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Google Colab

## Methodology

1. Data Cleaning
2. Feature Engineering
3. Data Merging
4. Exploratory Data Analysis
5. Trader Segmentation
6. Predictive Modeling (Bonus)

## Results

- Compared trader performance across market sentiment categories.
- Identified behavioral patterns among different trader segments.
- Proposed actionable trading strategies.
- Developed a Random Forest model to predict trade profitability.

## How to Run

1. Open the notebook in Google Colab or Jupyter Notebook.
2. Install the required libraries:

```bash
pip install pandas numpy matplotlib seaborn scikit-learn
```

3. Upload the datasets:
   - fear_greed_index.csv
   - historical_data.csv

4. Run all notebook cells sequentially.
